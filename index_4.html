Order of operations

Three tickets, one kitchen

Each part solves what the previous one couldn't reach — from static metadata, to a live browser session, to language-model reasoning over what was actually said.

PART 01 / 03

Metadata & Baseline Sentiment

Instaloader pulls public post and account metadata with no login required. TextBlob scores a pre-exported comment set and everything compiles into an 8-sheet Excel report.

instaloader textblob pandas openpyxl
→ sozal_foods_FULL_REPORT.xlsx
PART 02 / 03

Selenium Deep-Scroll Scraper

A real Chrome session scrolls the feed and opens each reel to reach comment threads static scraping can't see. A custom 3-tier engine classifies tone from keywords, slang and emoji.

selenium chrome webdriver rules-based NLP
→ 217 comments classified
PART 03 / 03

LLM Content Classification

Gemini 2.5 Flash labels each reel across 8 content-strategy dimensions; Whisper transcribes sampled audio to check captions against what's actually spoken.

gemini 2.5 flash whisper ffmpeg
→ sozal_classified_output.csv

Key discovery

Audio transcription check · 5 sampled reels

Whisper-transcribed audio from all five sampled reels matched the written captions almost word for word. @sozal_foods writes complete recipes and instructions directly into captions — which means caption-only analysis is a reliable proxy for spoken content, and Part 3 didn't need to transcribe the full catalogue to classify it accurately.

✓ Captions ≈ spoken audio, 5/5 sampled reels
